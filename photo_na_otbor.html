Фото на отбор

Вставляйте номера фото с комментариями по обработке и экспортируйте данные

Инструкция:

  • Кликните на заголовок для добавления модели товара
  • Кликните на ячейку для добавления номера фото
  • Нажмите "Экспорт в TXT" для сохранения данных в текстовый файл
  • Нажмите "Копировать в буфер" для копирования данных в буфер обмена
  • Нажмите "Сбросить таблицу" для очистки всех данных
  • При экспорте пустые строки и ячейки игнорируются
