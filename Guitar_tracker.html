Руководство пользователя

Добро пожаловать в Guitar Tracker — приложение для учёта работ по обслуживанию ваших гитар. Вы можете хранить записи о произведенных работах по каждой гитаре, каждого клиента вашей мастерской. Записи по обслуживанию отображаются для каждой гитары отдельно

  • Добавление гитары: нажмите «добавить», укажите название и модель.
  • Выбор гитары: кликните по карточке и вызовите контекстное меню ПКМ.
  • Обслуживание: ПКМ по гитаре → «Добавить запись».
  • Удаление записи: ПКМ по строке записи — удалить.
  • Импорт/экспорт: сохраните или загрузите нового пользователя.
  • Имя пользователя: отображается в шапке — его можно редактировать.

Примечание: записи в истории отображаются в обратном хронологическом порядке (свежие — сверху).
